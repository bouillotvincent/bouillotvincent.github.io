Projet des vacances : Création d'un site internet

Objectif : apprendre à manipuler HTML et CSS

Méthode : Pendant ces vacances, vous devez réaliser un site internet !

Pour réaliser ce travail, vous devez apprendre à utiliser deux langages que sont HTML et CSS. Pour cela, vous allez consulter les pages bases en HTML et bases en CSS de la fondation Mozilla. Vous utiliserez l'éditeur Brackets (ou similaires) afin de coder en HTML et CSS.

À partir des menus déroulants sur la gauche de la page de Mozilla, vous êtes libres de découvrir le HTML et le CSS aussi loin que vous le souhaitez mais n'étudiez pas le Javascript !!

Un exemple, volontairement imparfait, de ce qui est demandé est disponible ici. Vous pouvez étudier (et vous inspirer) du code de cette page en cliquant sur "Afficher, Options pour les développeurs, CodeSource".

Travail à réaliser : Votre site doit impérativement comprendre :
  • Une page d'accueil appelée index.html contenant :
    • Une petite présentation académique (classe, spécialités, idées d'orientation...)
    • Une petite présentation personnelle (centres d'intérêts, sports pratiqués ...)
    • Une image (photo de vous ou autre)
    • Une liste de navigation permettant d'accéder à la page projets.html
  • Une page appelée projets.html contenant :
    • Une description du programme Python permettant de tracer le triangle de Sierpinski
    • Un accès à votre code Python grâce à un lien (de téléchargement)
    • Une liste de navigation permettant de revenir à la page index.html
  • Un fichier CSS style.css permettant de personnaliser le style de la page.

Procédure :

La procédure de mise en ligne décrite ci-dessous doit être rigoureusement suivie. Ne mettez pas de caractères accentués lors de la création de votre nom d'utilisateur.

  1. S'inscrire sur GitHub.com avec comme nom d'utilisateur : nomprenom (votre nom, votre prenom). Validez immédiatement la création de votre compte.
  2. Se connecter sur votre compte GitHub et faire les premiers réglages demandés.
  3. Créez un nouveau "Create a new Repository" et appelez-le nomprenom.github.io.
  4. Choisissez "Add a README file"
  5. Choisissez ensuite "creating a new file" comme indiqué ci-dessous :
  6. Dans votre fichier, écrivez < h1> Je suis en ligne ! < /h1> et appelez ce fichier index.html
  7. Cliquez sur "Settings" puis descendez jusqu'à trouver "Github Pages". Dans, "Branch", sélectionnez "main" puis "save"
  8. Vérifiez que votre site est bien en ligne à l'adresse 'nomprenom.github.io'. Lors de la création du site, cela peut prendre jusqu'à 20 minutes. Soyez patient.
  9. Maintenant que vous avez vérifié que votre site web fonctionne, vous vous pouvez déposer tous les fichiers HTML, CSS, jpeg, python... que vous avez créé sur votre ordinateur.
